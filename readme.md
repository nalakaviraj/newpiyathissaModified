## About NAVI POS

NAVI POS is a POS application by [NAVI Fosters](http://NAVIfosters.com), a brand of [The Web Fosters](http://thewebfosters.com).

## Installation & Documentation
You will find installation guide and documentation in the downloaded zip file.
Also, For complete updated documentation of the NAVI pos please visit online [documentation guide](http://NAVIfosters.com/NAVI-pos/).

## Security Vulnerabilities

If you discover a security vulnerability within NAVI POS, please send an e-mail to support at thewebfosters@gmail.com. All security vulnerabilities will be promptly addressed.
